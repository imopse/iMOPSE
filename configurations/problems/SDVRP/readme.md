## Test Instances

The experiments use selected VRPSD benchmark instances with the following Best Known Solutions (BKS):

- **SD3** (16 cities) – BKS: 43,060
- **eil22** (22 cities) – BKS: 375
- **SD8** (48 cities) – BKS: 506,828
- **S51D4** (50 cities) – BKS: 1,551
- **SD14** (100 cities) – BKS: 1,071,550
- **SD18** (160 cities) – BKS: 1,419,369

**Data source:**  
DIMACS VRP Challenge – VRPSD  
http://dimacs.rutgers.edu/programs/challenge/vrp/vrpsd/
